# Registration and Sign-in document

## Use Cases

![diagram of the use cases for the UI of the homepage marketplace system](./_assets/d3.png)

### Logout

The user should be able to logout through an interaction at the top of the menu ribbon which will bring them back to the login/register page.

### View Statuses of all Active Auctions

The user is able to scroll until the max amount of active auctions on the homepage are viewed. The user should be able to see the first picture of the item, its price, its current bid, and how much time is left to bid on it. 

#### View Purchased Items

If an item is purchased by the user's tokens, a shopping bag icon can be interacted with and redirect the user to the items that they have purchased.
